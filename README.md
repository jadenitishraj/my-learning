# 📚 ASCII Knowledge Trees — High-Signal Learning

This repository is built on a simple idea:

> Most content is noise. We only keep what matters.

Instead of saving links or long notes, this project captures knowledge as **ASCII trees** — structured, minimal, and easy to scan.

---

## 🚀 What This Repo Does

* Converts **any knowledge source** into ASCII trees
* Organizes them into domain-specific folders
* Uses community review to keep quality high

Sources can include:

* YouTube videos
* Books
* Research papers
* Blogs / articles
* Courses

AI-related content is especially encouraged, but not required.

---

## 🤝 Contribution Guidelines

1. Pick a **high-value source** (any format)
2. Create a **clean ASCII tree**
3. Add it to the correct folder
4. Open a Pull Request

### Merge Criteria

* High signal, low noise
* Clear structure
* Easy to scan
* No fluff
* Community validation (reviews / upvotes)

Low-quality or verbose trees will be rejected.

---

## 🧠 Philosophy

* Focus on **20% that matters**
* Ignore **80% noise**
* Prefer **structure over summaries**
* Build a **shared knowledge system**

---

## 📌 Rules

* No long explanations
* No duplication
* Keep naming consistent
* One concept per tree
